# Space Shooter Deluxe

A simple yet visually enhanced 2D spaceship shooting game built using **Java Swing and AWT**.

---

## Description

Space Shooter Deluxe is a classic arcade-style game where you control a spaceship to shoot down alien enemies before they reach the bottom of the screen. Enjoy smooth animations, a starry space background with twinkling stars, glowing bullets, and a cool alien enemy design.

---

## Features

- Smooth spaceship movement (left and right)
- Shoot glowing bullets with the spacebar
- Enemy ships descend steadily with alien-style design
- Twinkling starfield background with gradient space colors
- Score tracking with shadowed text
- Game Over screen with the option to restart by pressing **ENTER**
- Responsive controls with keyboard input

---

## Controls

| Key          | Action                |
| ------------ | --------------------- |
| Left Arrow   | Move spaceship left   |
| Right Arrow  | Move spaceship right  |
| Spacebar     | Shoot bullets         |
| Enter (after game over) | Restart game |

---

## How to Run

1. Ensure you have **Java 8** or higher installed.
2. Download or clone the repository.
3. Compile the game:

   ```bash
   javac SpaceShooter.java
